Introdução

Aqui apresentamos um aplicação em Angular que se conecta à um banco de dados relacional através de uma API RESTful. Como entidade a ser persistida, ou seja, os dados que serão gravados no banco, usaremos um Lembrete composto por uma mensagem de texto, bem como um nível de prioridade, com valores pré-definidos (alta, média ou baixa). Para criar esse aplicativo utilizamos o Angular em sua versão 7.
