Visão geral

Este é o modulo principal da nossa aplicação, criado automaticamente pelo Angular. Nesse módulo definimos todos os componentes e módulos que estarão disponíveis para serem utilizados pela aplicação.

Comentários

Linha 1: Importação obrigatória para funcionamento da aplicação. É esse módulo que definirá como a aplicação deve ser executada. 

import [curly_open] BrowserModule [curly_close] from '@angular/platform-browser';

Linha 2: O módulo NgModule permitirá o uso de um decorator que indica pro Angular que a classe AppModule se trata de um módulo. 

import [curly_open] NgModule [curly_close] from '@angular/core';

Linha 3: Modulo utilizado para a criação do formulário de busca. Sem esse módulo não seríamos capazes de recuperar os valores digitados pelo usuário. 

import [curly_open] FormsModule [curly_close] from '@angular/forms';

Linha 4 a 12: Importamos todos os componentes e módulos criados para a aplicação. Isto é, o módulo de rotas, os componentes de aplicação, de listagem de lembretes, de cadastro e atualização de lembretes e mensagem de erro.
