Visão geral

Aqui está declarado o template do componente app. Podemos dividir esse template em duas partes. Na primeira delas criamos um cabeçalho que será exibido em todas as páginas da aplicação. Logo abaixo, inserimos o componente de rotas, o qual determina o local em que o Angular incluirá o conteúdo que corresponda a rota que está sendo atualmente navegada.

Comentários

Linha 1, 2 e 3: Criamos um nav com as classes navbar, navbar-light e bg-light do bootstrap para estilização.

Linha 4: Adicionamos o componente de rotas. Esse componente indica para o Angular que, para cada rota especificada no arquivo src/app-routing.module.ts, o componente referente a esta rota, ou componente de página, deve ser carregado aqui.
