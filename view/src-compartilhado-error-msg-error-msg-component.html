Visão geral

Este é o template do componente de erro. É aqui que definimos a aparência da mensagem de erro, bem como se esta deve ser exibida ou não.

Comentários

Linha 1: Criamos uma div com as classes alert e alert-danger do bootstrap. Essas classes darão uma aparência avermelhada ao elemento. Além disso, utilizamos o atributo *ngIf para verificar se existe valor atribuído ao atributo error da classe do componente. Caso verdadeiro, a mensagem será exibida. 

<div class="alert alert-danger" *ngIf="error">

Linha 2: Exibimos o erro definido pelo componente pai. 

[curly_open][curly_open] error [curly_close][curly_close]
